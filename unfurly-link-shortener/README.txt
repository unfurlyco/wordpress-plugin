Automatically shorten links in posts using unfur.ly service with custom domain support and UTM parameter management.

== Description ==
Unfurly Link Shortener automatically shortens links in your WordPress posts using the unfur.ly service. Features include custom domain support, UTM parameter management, and original URL preservation.

Key Features:
* Automatic link shortening in posts
* Custom domain support
* UTM parameter management
* Analytics tracking
* Original URL preservation
* Gutenberg editor integration

== Installation ==
1. Upload the plugin files to `/wp-content/plugins/unfurly-link-shortener`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Unfurly Shortener screen to configure the plugin

== Frequently Asked Questions ==
= Do I need an API key? =
No, but using one enables advanced features like analytics and custom domains.

= Can I use my own domain? =
Yes, with an API key you can configure a custom domain.
